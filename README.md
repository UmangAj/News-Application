# News Application

## Overview

The News Application is a dynamic web application that fetches and displays the latest news articles using the News API. It features intuitive search functionality and category navigation, providing users with a seamless experience to stay updated on various topics.

## Features

- **Real-Time News Fetching**: Displays the latest news articles in real-time using the News API.
- **Search and Navigation**: Users can search for news based on keywords and navigate through categories like IPL, Finance, and Politics.
- **Responsive Design**: The application features a responsive, card-based layout ensuring a consistent user experience across all devices.

## Technologies

- **Frontend**: HTML, CSS, JavaScript
- **API**: News API
- **JavaScript**: Asynchronous JavaScript (fetch API) for handling API requests and real-time UI updates

## Usage

- Use the search bar to find news articles based on specific keywords.
- Navigate through different categories (IPL, Finance, Politics) using the navigation menu.
- Click on any news card to read the full article in a new tab.

## Project Structure

- **index.html**: The main HTML file containing the structure of the application.
- **style.css**: The CSS file for styling the application.
- **script.js**: The JavaScript file for handling API requests and dynamic content updates.
- **logo.png**: The company logo used in the navigation bar.
